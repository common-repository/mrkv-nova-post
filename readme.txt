* Free shipping text if the order amount qualifies

== ğŸ‘‰ï¸ Getting started ==

1. [Register an account at Nova Post](https://novapost.com/form) 
2. Get your API key
3. Copy API key to plugin settings, and Save.
4. Add Nova Post shipping method to valid countries or regions. The plugin will display shipping method only in valid countries.

== Frequently Asked Questions ==

= Can I use the plugin without API key? =
No, the autocomplete will not work without valid and up-to-date API key.
= Will it work with checkout blocks? =
No, the plugin currently works only with default [woocommerce_checkout] shortcode.
